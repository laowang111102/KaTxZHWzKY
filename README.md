# 项目管理系统 V1.1 python439

## 项目概述

本项目是一款基于 Python 开发的项目管理系统软件，旨在为用户提供便捷的项目管理工具。当前版本仅支持 Windows 操作系统。

## 技术栈

- 开发语言：Python
- GUI 框架：PyQt/PySide
- 数据存储：SQLite

## 功能模块

- 项目创建与编辑
- 任务分配与管理
- 项目进度跟踪
- 数据统计与分析

## 系统角色

- 管理员：负责项目管理与系统设置
- 项目成员：负责执行项目任务

## 运行环境

- 操作系统：Windows
- Python 版本：3.x

## 目录结构

```
.
├── src
│   ├── main.py
│   ├── config.py
│   ├── models
│   │   └── data_model.py
│   └── views
│       ├── main_window.py
│       └── about_window.py
├── resources
│   └── images
└── README.md
```

## 部署步骤

1. 确保已安装 Python 3.x
2. 克隆或下载项目源码
3. 安装依赖：`pip install -r requirements.txt`
4. 运行主程序：`python main.py`
